Notes (TLPH 01/15): The Algebra Behind Types

1.1 Isomorphisms and Cardinalities

An isomorphism between two types a and b is the pair of morphisms

f :: a -> b
g :: b -> a

such that

f . g = id :: b -> b
g . f = id :: a -> a

a isomorphic to b can be written a ≅ b.

The cardinality of a type is the number of inhabitants of that type.

Two types with the same finite cardinality n are isomorphic, and there are n! unique isomorphisms between them.

1.5 Canonical Representations

The canonical representation of a type is as a sum of product types. That is, the product types in the canonical representation are fully distributed over the sum types.
